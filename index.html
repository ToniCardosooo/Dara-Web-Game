GAME RULES
  1. Each player has a set of 12 pieces of the same color, which are initially not placed on the game board.

  2. Every match has 2 phases:
    • Drop Phase
    • Move Phase

  3. Pieces can be grouped in 3 consecutive pieces of the same color, creating a line, both vertically and horizontally.

  4. Drop Phase
    • Players alternatively place a piece in a free cell of the board.
    • Pieces cannot create lines with size greater than 3.
    • There can be lines of 3 pieces.

  5. Move Phase
    • Each player moves alternatively one of their pieces.
    • Each piece can only be moved to the vertically or horizontally adjacent empty cells.
    • Creating a line of 3 pieces allows the current player to remove one and only one piece from the opponent.
    • It's not allowed to form lines with more than 3 pieces.
    • In the following round of the same player, its last moved piece cannot be played back to the cell where it came from.

  6. The match is finished when one of the following conditions is met:
    • A player only has 2 pieces left.
    • A player cannot make any valid move (i.e., all pieces are trapped)
    If player meets any of these conditions, then the opponent is the winner.
